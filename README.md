# spring-eslint
[![Downloads]()](https://www.npmjs.com/package/spring-eslint)

提供了多种eslint配置方案



### spring-eslint
基础js代码检查

```
npm install eslint eslint-plugin-import spring-eslint -D
```

### spring-eslint/vue
基础js代码检查+vue的代码检查
```
npm install eslint eslint-plugin-import spring-eslint eslint-plugin-vue -D
```
